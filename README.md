# 气象数据管理系统（Flask + MySQL）

基于Flask开发的企业内网气象数据管理系统，支持气象实况查询、后台任务管理、能见度预报模型管理三大核心功能，所有资源均支持本地加载（适配内网环境）。


## 📋 项目介绍
本系统为企业内网设计，解决气象数据可视化查询、任务调度管理及预报模型维护需求，核心特点：
- 本地资源加载：无外部依赖，CSS/JS/地图服务均本地部署；
- 数据可视化：基于OpenMapTiles实现中国区域地图+气象要素叠加显示；
- 权限控制：后台功能需登录访问，保障数据安全；
- 模块化设计：功能按模块拆分，便于维护和扩展。


## 🎯 功能模块
| 模块名称               | 核心功能                                                                 |
|------------------------|--------------------------------------------------------------------------|
| 气象实况数据查询       | 地区筛选、时间选择、气象要素（气温/降水/湿度等）显示、地图自适应、数据表格展示 |
| 后台任务管理           | 任务信息/作业参数的CRUD、任务运行状态查询、作业日志查看                   |
| 能见度预报模型管理     | 模型信息CRUD、模型任务同步（调用存储过程）、模型状态监控                 |


## 🛠️ 技术栈
| 类别       | 技术/工具                          |
|------------|-----------------------------------|
| 后端       | Python 3.8+、Flask 2.3+、Flask-SQLAlchemy |
| 前端       | HTML5、Bootstrap 5、OpenLayers 7、jQuery |
| 数据库     | MySQL 8.0+                        |
| 部署工具   | Git、Python Virtual Environment   |
| 环境管理   | python-dotenv（敏感信息管理）     |
